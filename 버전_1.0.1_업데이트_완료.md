# 한국어 프로그래밍 언어 v1.0.1 업데이트 완료

## 변경 사항

### 🐛 버그 수정
1. **파서 버그 수정**: 함수/조건문/반복문 본문에서 `보여줘` 문장이 제대로 인식되지 않던 문제 해결
2. **인터프리터 버그 수정**: 함수 반환값이 제대로 전달되지 않던 문제 해결

### 📝 수정된 파일
- `compiler/parser.py`: 블록 파싱 시 PRINT 토큰 종료 조건 추가
- `compiler/interpreter.py`: ReturnStatement 반환값 전파 로직 수정

## 설치 방법

### 새로 설치
```bash
pip install dist/korean_lang-1.0.1-py3-none-any.whl
```

### 업데이트
```bash
pip install --force-reinstall dist/korean_lang-1.0.1-py3-none-any.whl
```

## 테스트

### 피보나치 예제
```
함수 피보나치(n):
    만약 n <= 1:
        반환 n
    아니면:
        반환 피보나치(n-1) + 피보나치(n-2)

보여줘 "피보나치 수열:"
보여줘 피보나치(0)
보여줘 피보나치(1)
보여줘 피보나치(5)
보여줘 피보나치(10)
```

### 예상 출력
```
피보나치 수열:
0
1
5
55
```

[오류 발생] 'python' is not recognized...

이 오류는 Python이 설치되지 않았다는 의미입니다.

1단계: Python 설치 도우미 실행
   
   install_python_helper.bat 더블클릭
   
   → Microsoft Store 자동 열림
   → Python 3.12 설치
   → 완료!


2단계: IDE 실행

   run_ide.bat 더블클릭
   
   → IDE 시작!

방법 1: Microsoft Store (추천)
   1. Windows 키 누르기
   2. "Microsoft Store" 검색
   3. "Python 3.12" 검색
   4. "설치" 클릭

방법 2: 공식 웹사이트
   1. https://www.python.org/downloads/ 방문
   2. "Download Python" 클릭
   3. 설치 시 "Add Python to PATH" 체크!
   4. 컴퓨터 재시작
